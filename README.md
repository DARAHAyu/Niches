## Nichesについて

Nichesであなたのニッチな特技を活かして、お悩みを持つ人を助けてみませんか？

例えばNichesではこんなことができます。

- あなたの特技の掲示
- あなたの依頼の掲示
- メッセージ機能
- 特技/依頼の検索

## Nichesの使い方

Nichesの使い方は非常にシンプルです。

ユーザ登録を済ませてログインをしたら、特技または依頼の掲示をするだけ。

また、特技や依頼の検索をすることもできます。

実際に依頼主とダイレクトメッセージのやり取りをすることも可能です。

相互評価機能や通報機能もあるので、嫌がらせも心配もありません。
